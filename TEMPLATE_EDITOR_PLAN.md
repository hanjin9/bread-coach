# 호흡 영상 템플릿 제작 프로그램 (Canva 스타일)

## 🎯 프로젝트 목표
사용자가 직접 호흡 영상의 모든 요소(파이프라인, 구슬, 배경, 효과음, 호흡 패턴)를 편집할 수 있는 Canva 스타일의 드래그 앤 드롭 편집기 구축

---

## 📋 핵심 기능

### 1️⃣ 편집 인터페이스 (Canva 스타일)
- **좌측 패널**: 레이어 관리 (배경, 파이프라인, 구슬, 효과음)
- **중앙 캔버스**: 실시간 미리보기 (Canvas 기반)
- **우측 패널**: 속성 조절 (색상, 크기, 굵기, 곡률, 투명도)
- **하단 패널**: 호흡 패턴 타이밍 설정

### 2️⃣ 파이프라인 편집
- 굵기 조절 (1px ~ 50px)
- 시작/끝 지점 연장 (화면 끝까지)
- Bezier 곡선 제어점 (곡률 조절)
- 색상 커스터마이징 (Hex/RGB/그라데이션)
- 부드러운 굴곡 (각진 모서리 → 라운드)

### 3️⃣ 구슬 편집
- 크기 조절 (10px ~ 100px)
- 색상 & 그라데이션
- 질감 (광택, 매트, 메탈릭)
- 투명도 조절
- 그림자 효과

### 4️⃣ 배경 & 효과음
- 배경 이미지 업로드/선택
- 효과음 MP3 업로드/선택
- 볼륨 조절
- 페이드 인/아웃

### 5️⃣ 호흡 패턴
- 들숨/정지/날숨 초수 입력
- 반복 횟수 설정
- 실시간 타이밍 미리보기

### 6️⃣ 템플릿 관리
- 템플릿 저장 (클라우드)
- 프리셋 로드
- 영상 생성 (MP4 다운로드)

---

## 🛠️ 기술 스택

| 계층 | 기술 | 용도 |
| :--- | :--- | :--- |
| **프론트엔드** | React Native | 모바일 앱 UI |
| **편집 캔버스** | Canvas API + Skia | 실시간 렌더링 |
| **애니메이션** | Framer Motion | 부드러운 전환 |
| **상태 관리** | Zustand | 전역 상태 |
| **백엔드** | Node.js + Express | API 서버 |
| **영상 생성** | FFmpeg | MP4 생성 |
| **클라우드** | AWS S3 | 템플릿 저장 |

## 📁 파일 구조

```
src/
├── pages/
│   └── TemplateEditor.tsx          # 메인 편집 페이지
├── components/
│   ├── EditorCanvas.tsx             # 중앙 캔버스
│   ├── LeftPanel.tsx                # 좌측 레이어 패널
│   ├── RightPanel.tsx               # 우측 속성 패널
│   ├── BottomPanel.tsx              # 하단 타이밍 패널
│   └── PreviewPlayer.tsx            # 미리보기 플레이어
├── services/
│   ├── pipelineService.ts           # 파이프라인 로직
│   ├── orbService.ts                # 구슬 로직
│   ├── templateService.ts           # 템플릿 저장/로드
│   └── videoGeneratorService.ts     # 영상 생성
├── store/
│   └── templateStore.ts             # Zustand 스토어
└── types/
    └── template.ts                  # 타입 정의
```

## 🚀 구현 순서

1. **편집 인터페이스** - Canva 스타일 레이아웃
2. **파이프라인 편집** - 실시간 미리보기
3. **구슬 스타일** - 색상, 질감, 크기
4. **배경/효과음** - 업로드 및 관리
5. **호흡 패턴** - 타이밍 설정
6. **영상 생성** - FFmpeg 통합
7. **최종 검사** - 대시보드 연동
