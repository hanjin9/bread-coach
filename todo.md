# Bread Coach - 호흡 가이드 모바일 앱 개발 체크리스트

## 📱 프로젝트 개요
- **프로젝트명**: Bread Coach (호흡 코치 앱)
- **플랫폼**: 모바일 앱 전용 (iOS + Android)
- **기술 스택**: React Native + Expo
- **테마**: 다크 럭셔리 블랙 & 골드
- **배포 대상**: App Store, Play Store
- **결제**: 200개국 동시 결제 지원 (최종 단계)

---

## 🏗️ 1단계: 전체 뼈대 설계

### 데이터베이스 스키마
- [ ] 사용자 테이블 (users) - ID, 이메일, 휴대폰, 프로필, 구독 상태
- [ ] 호흡 템플릿 테이블 (breathing_templates) - 호흡 주기, 배경, 효과음, 파이프라인 설정
- [ ] 호흡 영상 테이블 (breathing_videos) - 템플릿ID, 생성 시간, 영상 URL, 상태
- [ ] 사용자 스케줄 테이블 (user_schedules) - 사용자ID, 영상ID, 예약 시간, 완료 여부
- [ ] 에셋 라이브러리 테이블 (assets) - 배경 이미지, 효과음 MP3, 메타데이터
- [ ] 관리자 설정 테이블 (admin_settings) - 기본 템플릿, 프리셋, 설정값
- [ ] 구독 및 결제 테이블 (subscriptions, payments) - 구독 유형, 결제 기록, 상태

### API 구조
- [ ] 인증 API (로그인, 로그아웃, 토큰 갱신)
- [ ] 호흡 영상 API (생성, 조회, 삭제)
- [ ] 스케줄 API (예약, 조회, 수정)
- [ ] 에셋 API (업로드, 조회, 삭제)
- [ ] 관리자 API (템플릿 생성, 설정 수정)
- [ ] 결제 API (결제 처리, 구독 관리)

### 아키텍처 설계
- [ ] 프론트엔드 폴더 구조 (screens, components, hooks, utils, services)
- [ ] 백엔드 폴더 구조 (routes, controllers, services, models, middleware)
- [ ] 상태 관리 (Zustand + TanStack Query)
- [ ] 로컬 스토리지 (AsyncStorage)
- [ ] 클라우드 스토리지 (S3 또는 Firebase Storage)
